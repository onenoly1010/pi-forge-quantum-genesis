How to contribute to the Pi Forge Quantum Genesis.

## Code of Conduct
*   Be respectful.
*   Be constructive.
*   Seek harmony.

## Pull Request Protocol
1.  **Fork** the repository.
2.  **Branch** from `main` (`feature/my-new-feature`).
3.  **Commit** your changes.
4.  **Test** locally.
5.  **Push** to your fork.
6.  **Open PR** against `main`.

## Style Guide
*   **Python:** PEP 8.
*   **JavaScript:** ES6+.
*   **Commit Messages:** Conventional Commits (e.g., `feat: add new resonance pattern`).

## The Veto
The **Veto Triad** applies to code contributions as well. Code that introduces dissonance or security risks will be rejected.
