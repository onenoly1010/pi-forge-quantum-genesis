# Ritual Calendar

> *Sacred ceremonies that maintain the rhythm of the Forge*

## Weekly Rituals

### Resonance Check-in (Wednesdays)
**Time**: 14:00 UTC  
**Duration**: 30 minutes  
**Purpose**: Review system health and address immediate concerns

**Agenda**:
1. Review resonance metrics from the past week
2. Discuss any guardian alerts triggered
3. Prioritize issues for the coming week
4. Quick wins and blockers

### Code Harmony Session (Fridays)
**Time**: 16:00 UTC  
**Duration**: 1 hour  
**Purpose**: Collaborative code review and knowledge sharing

**Agenda**:
1. Review outstanding pull requests together
2. Discuss architectural decisions
3. Share learning from the week
4. Plan for next week's development

## Monthly Rituals

### Guardian Rotation (1st of each month)
**Purpose**: Rotate primary guardian responsibilities

**Process**:
1. Outgoing guardian provides handoff notes
2. Incoming guardian assumes primary on-call
3. Update GUARDIANS.md with rotation
4. Archive previous month's metrics

### Covenant Alignment Review (15th of each month)
**Purpose**: Assess adherence to covenant principles

**Process**:
1. Review SEAL_LEDGER.json entries
2. Identify areas of strong alignment
3. Discuss areas needing improvement
4. Plan corrective actions if needed

---

## Quarterly Rituals

### Covenant Retrospective (End of each quarter)
**Purpose**: Deep review and potential amendments

**Process**:
1. Gather community feedback (2-week window)
2. Guardian discussion and proposal drafting
3. Community vote on amendments (7-day window)
4. Ratify changes and update version

### Transcendence Celebration
**Purpose**: Recognize achievements and contributions

**Agenda**:
1. Highlight major milestones achieved
2. Recognize outstanding contributors
3. Share success stories and learnings
4. Set intentions for next quarter

---
